AboutMe

Agentic AI Engineer | LLM Developer | iOS & Cloud Builder

Hello, I'm Nikhil Deekonda — an AI and cloud engineer passionate about building intelligent, multimodal applications. I completed my MS in Artificial Intelligence at Yeshiva University, NY, maintaining a 3.9 GPA.

My professional experience includes building agentic systems at Lumamind - REVIV AI, where I lead the development of an iOS-based virtual clinic and orchestrate multi-agent workflows using AWS Bedrock, FastAPI, Cognito, and LangChain.

At LTIMindtree, I developed and fine-tuned LLMs (GPT, Falcon, LLaMA-2, Mistral), engineered secure pipelines and validation flows, and deployed scalable solutions using AWS and LangGraph.

I specialize in deploying LLM-powered solutions with FastAPI, iOS integration, and real-time cloud orchestration. I believe in using AI responsibly to solve real-world problems and create human-centered experiences.

MyProjects

TaskFin: Agent-Based Financial Automation

Developed a multi-agent LLM-powered bill payment system using LangChain + ReAct. It features authentication, state-tracking, and orchestration agents running in a conversational Streamlit UI. MFA simulation and synthetic APIs enable real-world task automation in secure environments.

LangChain Mistral Claude Streamlit LLMs

LungAware: AI Lung Cancer Detection App

Built an Android/iOS app using Swift and Kotlin integrating a CNN model for lung cancer prediction from CT scans. Integrated Grad-CAM for interpretability and deployed the model using TFLite for mobile inference.

Swift Kotlin CNN Grad-CAM TFLite

LLM Reliability Enhancer: Fact-Checking with DeBERTa

Fine-tuned a Microsoft DeBERTa-v3 model as a fact-checking cross-encoder. Integrated it into QA pipelines to validate answers in real time, reducing hallucinations. Achieved an F1 score of 85% on the Facebook Fact Checking Dataset.

DeBERTa Hugging Face Cross-Encoder Fact-Checking

Bird Sound Denoising with Custom Segmentation

Engineered a deep learning architecture to denoise bird sound spectrograms using visual segmentation techniques. Outperformed SOTA approaches, achieving an IoU of 66.24. Aimed at enhancing ecological monitoring.

Encoder-Decoder Deep Learning Segmentation Bioacoustics

AI-Powered Assistive Communication (FSR402)

Built a system using FSR402 pressure sensors and ML classifiers to assist semi-paralyzed users by interpreting finger gestures. Translated gestures into digital text using real-time signal processing pipelines in Python.
